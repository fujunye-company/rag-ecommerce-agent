## 3. 之后调用 Claude Code 的固定 prompt 前缀

```
只读模式。不要创建/修改/删除任何文件，不要Write/Edit/Bash。
只做Read/Glob/Grep分析。如需改文件，输出建议由Hermes执行。
```

**关键经验**：前缀必须 ≤3 行。过长（100+ 行）会导致 120s+ 无输出。

## 4. Claude Code 可用工具确认

| 工具 | 状态 | 验证方式 |
|------|------|---------|
| Read | ✅ 可用 | 连通性测试：读取 main.py，输出与实际文件一致 |
| Glob | ✅ 可用 | 连通性测试：列出 10 个 .md 文件，与实际一致 |
| Grep | ✅ 可用 | 会话日志确认可用 |
| Write | ❌ 静默失败 | 输出声称成功但文件不存在 |
| Edit | ❌ 未验证 | 按不可用处理 |
| Bash | ❌ 禁止 | 不用于双 Agent 嵌套 |
| WebSearch | ❌ 禁止 | 不作为可靠工具 |

**CC 调用耗时**：简易任务 ~54s，代码分析任务 ~130s
