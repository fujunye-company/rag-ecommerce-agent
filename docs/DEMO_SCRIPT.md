# 演示脚本 [全量]

> 5 个典型导购场景演示话术和备用方案

## 场景 1: 预算推荐
**输入**: "学生党，预算300以内推荐个耳机"
**预期**: 返回 2-3 个耳机商品卡片 + 推荐理由

## 场景 2: 参数咨询
**输入**: "这个音箱适合户外跑步用吗"
**预期**: 解释防水/续航/便携等参数

## 场景 3: 商品对比
**输入**: "对比一下 A 和 B 两款平板"
**预期**: 多维度对比表

## 场景 4: 图片导购
**输入**: [上传商品图片] "有类似这个的推荐吗"
**预期**: 识别商品特征 + 推荐同款

## 场景 5: 多轮对话
**输入**: "推荐个耳机" → "有便宜点的吗" → "这个续航多久"
**预期**: 上下文保持, 偏好更新
