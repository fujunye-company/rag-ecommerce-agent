# 演示脚本 — AI 全栈挑战赛

> 时长：3-5 分钟 | 场景覆盖：9 级 | 状态：场景 1-4+6 可演示，5/9 待实现

---

## 场景 1：单轮推荐 ✅ 可演示

**话术**：
> "推荐一款降噪耳机，预算2000以内"

**实际效果**（curl 验证通过）：
- Agent 识别意图 → Qdrant 检索 → SSE: text_delta + 5 product_cards + done
- 卡片含 image_urls、价格、评分、匹配度

---

## 场景 2：条件筛选 ✅ 可演示

**话术**：

**预期**：
- 结构化参数提取（price_max=500）+ 范围过滤
- 流式文本 + 商品卡片

## 场景 3：多轮追问 ✅ 可演示
