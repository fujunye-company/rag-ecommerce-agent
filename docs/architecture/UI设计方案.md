### 1.2 逐维度分析

|------|:---:|---------|--------|

| **文心一言** | 7 | 流式输出稳定，但偶有气泡重绘闪动 | — |

## 2. 最佳UI模式提取

### 2.1 对话流模式 (气泡样式/打字动画/加载态)

```
┌─ 来源：豆包 + ChatGPT ──────────────────────────────────┐
│                                                          │
│  流式输出模式 (Token-level SSE Streaming):                │
│  ┌──────────────────────────────────────────────────┐   │
│  │ 后端 SSE 事件流:                                  │   │
│  │   text_delta  → {"content": "为", "seq": 1}       │   │
│  │   text_delta  → {"content": "您", "seq": 2}       │   │
│  │   text_delta  → {"content": "推", "seq": 3}       │   │
│  │   ...           (逐 token 推送)                   │   │
│  │   product_cards → {"products": [...]}             │   │
│  │   done        → {"session_id": "xxx"}             │   │
│  └──────────────────────────────────────────────────┘   │
│                                                          │
│  前端渲染策略:                                           │
│  ┌──────────────────────────────────────────────────┐   │
│  │ 1. 文本队列: StateFlow<String> 增量拼接            │   │
│  │ 2. 光标动画: 闪烁 "▌" 块光标 (豆包特色)            │   │
│  │ 3. 打字节奏: AnnotatedString 构建, 无额外延迟       │   │
│  │ 4. 自动滚动: LazyColumn.animateScrollToItem(last)  │   │
│  │ 5. 卡片延迟: text_delta流结束后再渲染 product_cards │   │
│  └──────────────────────────────────────────────────┘   │
│                                                          │
│  气泡样式 (参考豆包):                                    │
│  - 用户气泡: 右对齐, primaryContainer色, 14dp圆角       │
│  - AI气泡:   左对齐, surfaceVariant色, 14dp圆角         │
│  - 卡片气泡: 左对齐, 白色卡片嵌入, 含动态高度           │
│  - 间距: 相邻同角色气泡间 4dp, 不同角色间 12dp          │
│                                                          │
│  加载态动画:                                             │
│  - 文本加载: 三个跳动点 "..." (类似豆包)                 │
│  - 卡片加载: Shimmer效果骨架屏 (参考淘宝问问)            │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

## 附录B：设计决策记录

| 决策 | 选项A | 选项B | 选择 | 理由 |
|------|-------|-------|:---:|------|
| 卡片滚动方向 | 横向(LazyRow) | 纵向(LazyColumn) | A | 横向更紧凑, 每行3-4张, 与淘宝问问/文心一言一致 |
| 流式光标样式 | 闪烁▌块 | 跳动●点 | A | 豆包已验证, 代码提示感强 |
| 文本+卡片发送策略 | 混在一起 | 分开发送(text_delta→product_cards) | B | 后端独立事件, 前端渐进渲染, 体验更好 |
| 对比页进入方式 | 底部弹出 | 右侧推入 | A | 对比是辅助操作, 底部面板更轻量 |
| 暗色模式 | 纯黑#000 | 深灰#121212 | B | Material3推荐, 避免AMOLED纯黑刺眼 |
| 导航结构 | 单页全对话 | 3Tab底部导航 | B | 历史会话独立入口更清晰, ChatGPT模式验证 |
